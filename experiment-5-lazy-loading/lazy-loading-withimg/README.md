# Lazy Loading with Image

This project demonstrates image lazy loading combined with immediately loaded text content.

## Features
- Lazy loaded images with loading state
- Text content loaded immediately on the right side
- "Please wait" message shown while image is loading
- Responsive layout using Flexbox

## Installation
```bash
npm install
```

## Development
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Preview
```bash
npm run preview
```
